# Design Department

**Call this department when you need visual design, UX, or creative direction** — building UI components, establishing brand identity, designing user experiences, crafting visual narratives, injecting personality into interfaces, generating inclusive imagery, or optimizing for accessibility. Also for microcopy, tone-of-voice, and content strategy.

**Leader**: Brand Guardian
**Model tier**: Members = Sonnet, Leaders = Opus

## Members

| Agent | What it does |
|---|---|
| Brand Guardian | Brand consistency, visual identity, creative direction |
| UI Designer | Visual design systems, component libraries, pixel-perfect interfaces |
| UX Researcher | User behavior analysis, usability testing, data-driven insights |
| UX Architect | UX systems, CSS architecture, implementation guidance |
| Visual Storyteller | Multimedia content, brand storytelling, visual narratives |
| Whimsy Injector | Playful, delightful interactions that differentiate brands |
| Image Prompt Engineer | Crafting prompts for AI image generation |
| Inclusive Visuals Specialist | Culturally accurate, bias-free imagery and representation |

## Skills

These skills are loaded as process gates by dept members:
- **figma-ui-ux-consistency** — end-to-end Figma-to-code consistency lifecycle: audit → foundations → component library → implementation → governance
- **ui-ux-pro-max** — advanced UI/UX design, design systems, interaction patterns
- **frontend** — implementation of design specs, responsive layouts
- **copywriting** — microcopy, tone of voice, content strategy
