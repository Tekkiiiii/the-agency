## 1. Visual Hierarchy

**Core rule:** If the eye leaves the page before reaching the CTA, the design has failed regardless of how beautiful it is.

### Three Dominance Tiers
- **Tier 1 — Dominant (60-70% visual weight):** The single most important element. Usually the headline or hero image. Must win every visual competition on the page.
- **Tier 2 — Sub-dominant (20-30% visual weight):** Supporting elements (subheadline, social proof, product image). Reinforces Tier 1 without competing.
- **Tier 3 — Subordinate (10% visual weight):** Body copy, captions, fine print. Legible but never draws the eye from Tier 1 or 2.

### Five Hierarchy Tools
- **Size:** Headline must be largest. 3:1 ratio minimum — header to body text.
- **Contrast:** Dark on light OR light on dark. Never mid-tones against mid-tones.
- **White space:** Space around an element increases perceived importance. CTA: minimum 16px padding, 24px preferred.
- **Proximity:** Related elements cluster. Unrelated elements separate. Gestalt proximity = brain-readable grouping.
- **Position:** Top-left starts the eye journey (LTR cultures). Critical elements belong above the fold.
- **Directional cues:** Arrows, faces looking at CTA, flow lines guide the eye to the conversion point.

### 3-Second Rule
A viewer decides in 3 seconds whether to engage. The dominant element must communicate the core value proposition within that window — no scrolling, no reading, no effort required.

## 2. Color Theory

### 60-30-10 Rule — Non-negotiable for all marketing assets
- **60% — Dominant brand color:** Backgrounds, large surfaces.
- **30% — Secondary color:** Sections, cards, supporting areas.
- **10% — Accent color:** CTAs, highlights, key data points ONLY. The accent color loses meaning when overused.

### Color Psychology for CTAs
- **Warm (red, orange, yellow):** Urgency, energy, promotions, limited-time offers.
- **Cool (blue, green):** Trust, calm — effective for B2B, finance, healthcare.
- **CTA winner:** Orange and green outperform red in most A/B tests (red signals stop/danger in many contexts).

### Hard Contrast Rules
| Rule | Standard | Why it matters |
|------|----------|----------------|
| Minimum contrast ratio | 4.5:1 (WCAG AA) | Legal compliance + readability for all users including low vision |
| Avoid pure black on pure white | Use #1a1a1a on #F8F8F8 | Pure #000 on #FFF causes eye strain over reading duration |
| Text over images | Always add overlay or shadow | Background images behind text almost always fail contrast checks |
| Brand color drift | Always use exact hex codes | Eyeballing colors erodes brand equity |

### Reliable Contrast Pairs
- Navy + White — B2B trust anchor
- Charcoal + Yellow — Energy + authority
- Forest Green + Cream — Premium, organic feel
- Deep Burgundy + Gold — Luxury positioning
- Off-black + Off-white — Universal readable baseline

## 3. Typography

**Max 2 typefaces per asset.** A display/serif for headlines and a sans-serif for body. Every additional typeface adds cognitive load and weakens brand recognition.

### Scale and Sizing
| Element | Size | Weight | Notes |
|---------|------|--------|-------|
| Ad headline / Hero title | 36-48px+ | Bold (700) | Must win the hierarchy. 3:1 ratio to body. |
| Section headline (web) | 24-36px | Bold (700) | Display or serif preferred |
| Subheadline | 18-22px | Semi-bold (600) | Sans-serif; bridges headline and body |
| Body copy | 16-18px | Regular (400) | **Never below 16px for screen-read content** |
| Labels / captions / badges | 12-14px | Bold (700) | All-caps acceptable at this size only (max 4 words) |
| Fine print / legal | 10-12px | Regular | Never below 10px; must still meet contrast minimums |

### Readability Rules
- **Line height:** Body copy 1.5x font size minimum. Headlines 1.1-1.25x.
- **Line length:** Desktop 60-80 characters per line. Mobile 30-40 characters.
- **All-caps rule:** All-caps only for labels/badges of 4 words or fewer. Long all-caps reduces reading speed by up to 13%.
- **Weight contrast:** Bold headlines next to Regular body creates visual rhythm without a second typeface. Skip Medium weight — creates neither hierarchy nor body-text comfort.

## 4. Layout Principles

### Scan Patterns
- **F-Pattern — Long-form / Web pages:** Users make two horizontal sweeps across the top, then a vertical drop down the left edge. Content in the right half of long paragraphs is largely invisible. Design response: key information at F-pattern entry points, subheadings, bolding, and bullets.
- **Z-Pattern — Ads and sparse layouts:** Top-left → top-right → diagonal → bottom-left → bottom-right. Design response: logo top-left, headline top-center/right, key visual middle, CTA bottom-right.

### 8-Point Grid System — Mandatory for all agency assets
All padding, margins, and spacing values must be multiples of 8px (8, 16, 24, 32, 48, 64, 96px). Inconsistent spacing is the most common amateur mistake and the fastest way to signal unprofessional design. Random spacing destroys Gestalt proximity grouping.

### Rule of Thirds
Divide canvas into 3x3 grid. Place key elements at intersection points — not dead center. Centered compositions look static unless intentionally symmetrical for authority positioning.

### Gestalt Principles Applied
| Principle | Marketing application |
|-----------|----------------------|
| Proximity | Related content clustered (price + CTA together). Unrelated separated by meaningful whitespace. |
| Similarity | All CTAs same color + style. All body text same family. Pattern = expectation = trust. |
| Continuation | Arrows, curves, lines of sight direct eye toward CTA. |
| Closure | Partial shapes suggest completion — useful for progress indicators and tease reveals. |
| Figure/Ground | Product on a clean background. Subject must separate clearly from context. |
| Common Fate | Elements moving same direction perceived as a group — applies to animated CTAs. |

## 5. Conversion-Oriented Design

### Above-the-Fold Requirements
Everything the user needs to make a conversion decision must be visible without scrolling:
- Benefit-led headline (what the user gets, not what you do)
- Subheadline that answers "how" in one sentence
- Visual proof (product image, result screenshot, or social proof number)
- Primary CTA button — visible, high-contrast, action verb + outcome

### CTA Design Rules
**Button specs:**
- Minimum touch target: 44px x 44px (mobile accessibility)
- Minimum padding: 16px horizontal, 12px vertical
- CTA breathing room: 24px whitespace on all sides
- Maximum 2 CTAs per page — more dilutes focus
- Use brand's accent color exclusively for primary CTA

**CTA copy:**
- Action verb + outcome: "Start Free Trial" beats "Submit"
- First person: "Get My Report" outperforms "Get Your Report" in most tests
- Avoid generic verbs: Submit, Click Here, Go, Enter
- CTA copy and button color must contrast at 4.5:1

### Social Proof Placement (by conversion impact)
1. Faces + full names of real customers — highest trust signal
2. Anonymized testimonials with company or role
3. Logo strips of known clients
4. Aggregate numbers ("3,000+ companies trust us")
5. Security badges and guarantees — reduce perceived risk below primary CTA

### Encapsulation Technique
Place the lead-generation form or primary CTA inside a visually distinct container (card, box, arch frame, or contrasting background). This creates tunnel-vision effect. Removing distractions (e.g., top navigation on landing pages) alone has shown 20% conversion lifts.

### Banner Blindness
Users almost never look at anything that looks like an advertisement — even when relevant and well-designed. Ads that look like native site components get significantly more visual fixations. Design ads to blend with the destination environment.

## 6. Social Media and Carousel Design

### Format Specifications
| Format | Dimensions | Notes |
|--------|------------|-------|
| Instagram/LinkedIn carousel (square) | 1080 x 1080px | Default carousel format |
| Instagram/LinkedIn carousel (portrait) | 1080 x 1350px (4:5) | More screen real estate — higher visibility |
| Instagram/TikTok Stories/Reels | 1080 x 1920px (9:16) | Vertical video |
| Facebook feed post | 1200 x 630px | Link preview standard |
| Twitter/X | 1200 x 675px (16:9) | In-tweet image display |
| LinkedIn single image | 1200 x 627px | Feed display standard |

### Carousel Structure — 6-Slide Model
- **Slide 1 — Hook:** Single bold claim with curiosity gap. 6 words maximum. High-contrast background. Must work as standalone ad.
- **Slides 2-5 — Value Delivery:** One idea per slide. Left-to-right reading flow. Visual continuity: same palette, font, corner rounding. Bold key phrase visible at thumbnail size.
- **Slide 6 — CTA:** Clear next step. Repeat brand identity. Contact/link/handle. Urgency or incentive if appropriate.

### Text Coverage Rule
Maximum 20% of slide area covered by text. Instagram algorithm penalizes text-heavy images. Visual weight should dominate — text reinforces, not carries, the message.

### Social-Specific Rules
- Design for mobile-first — most social content consumed on phone screens
- Safe zones: keep critical text 80px from all edges (platform cropping risk)
- Font minimum 40px on social graphics — readable at mobile thumbnail size
- Test designs at thumbnail size (150px) before finalizing
- Swipe continuity: matching background bands at slide edges encourage swiping

## 7. Brand Consistency

**Why it compounds:** McKinsey: consistent brand presentation increases revenue up to 23%. Lucidpress: brand consistency increases revenue by 33%. Consistency is a business multiplier, not aesthetic preference.

### The 6-Layer Brand Identity System
1. **Visual Identity:** Logo (all variants), color palette (exact hex), typography stack, iconography style, illustration style.
2. **Verbal Identity:** Brand voice, tone variations by context, vocabulary list, words to avoid.
3. **Layout System:** Grid system, spacing scale, composition rules, safe zones, hierarchy model.
4. **Photography Style:** Curated image library, composition guidelines, color treatment, subject matter rules.
5. **Motion Language:** Animation easing, transition speed, kinetic typography rules, entrance/exit behaviors.
6. **Multi-Asset Templates:** Pre-built templates per format that lock Layers 1-5 in.

### Brand Consistency Failures — The 6 Common Failures
| Failure mode | What it looks like | Fix |
|---|---|---|
| **Color drift** | Using "close enough" colors when exact hex isn't remembered | Store hex codes in brand file; never eyeball |
| **Font substitution** | System fonts appearing when brand fonts aren't installed | Embed fonts or maintain a font kit; verify on export |
| **Logo distortion** | Squishing or stretching logos to fit a space | Always constrain proportions; create variants for different aspect ratios |
| **Rogue gradients** | Adding gradients to flat brands, or vice versa | Follow brand guidelines; when in doubt, stay flat |
| **Spacing inconsistency** | Different padding/margin values across materials | Enforce 8-point grid across all templates |
| **Off-brand photography** | Stock photos that feel emotionally misaligned with brand voice | Maintain curated approved image library; reject anything that "almost fits" |

### Agency Workflow: Consistency Across Clients
- Each client brand gets a dedicated `brand-guidelines.md` — single source of truth
- Templates are per-brand, never shared across clients without explicit color/font overrides
- Design QA always includes: color hex verification, font rendering check, logo aspect ratio check
- Client materials are never "inspired by" another client — visual cross-contamination erodes both brands

## 8. Common Design Mistakes (The Amateur Mark List)

These patterns signal inexperience and actively reduce conversion rates. Design QA must catch all of these before delivery.

1. **Inconsistent spacing** — Random padding/margin values destroy Gestalt proximity. Fix: 8-point grid, no exceptions.
2. **Competing visual weights** — Headline and image at equal size = no hierarchy = no decision direction. Dominant element must win at 60-70%.
3. **Low contrast text** — Text below 4.5:1 contrast fails WCAG AA. Mid-tone on mid-tone is invisible at a glance. Most common readability failure.
4. **Wall of text (F-pattern trap)** — Long unformatted paragraphs trigger F-pattern scanning. Break up with subheadings, bullets, and bold phrases.
5. **Weak CTA copy** — "Submit" and "Click Here" are conversion killers. CTAs need action verbs and outcome language.
6. **Too many CTAs** — More than 2 CTAs dilutes focus and creates decision paralysis.
7. **Overcrowded layouts** — One message per ad. One value point per carousel slide. One decision per page.
8. **Ad "look"** — Ads that announce themselves as ads are ignored. Native-feeling content gets more attention.

## 9. Design QA Checklist (Pass/Fail)

Use before any marketing asset is approved for delivery. Any unchecked item is a HIGH blocker.

### Visual Hierarchy
- [ ] Dominant element holds 60-70% visual weight
- [ ] Headline is the largest text element
- [ ] CTA is visually distinct from all other elements
- [ ] Eye journey leads to CTA without escape
- [ ] No two elements compete at equal visual weight

### Color
- [ ] Brand hex codes used (not approximations)
- [ ] 60-30-10 ratio maintained
- [ ] All text passes 4.5:1 contrast minimum
- [ ] CTA uses accent color exclusively
- [ ] No pure #000 on pure #FFF

### Typography
- [ ] Maximum 2 typefaces used
- [ ] Body text minimum 16px
- [ ] Line height minimum 1.5x for body
- [ ] Line length 60-80 chars desktop, 30-40 mobile
- [ ] All-caps only for labels, 4 words max
- [ ] Brand font rendering verified (not system font fallback)

### Layout
- [ ] 8-point grid applied to all spacing
- [ ] F-pattern or Z-pattern respected for asset type
- [ ] Critical content above the fold (no scroll required)
- [ ] CTA has 24px breathing room on all sides
- [ ] No random floating elements

### Social Media
- [ ] Correct dimensions for target platform
- [ ] Text covers less than 20% of slide area
- [ ] Fonts minimum 40px at design size
- [ ] Safe zones (80px) respected at all edges
- [ ] Carousel: slide 1 works as standalone ad
- [ ] Verified at thumbnail size (150px)

### Brand Consistency
- [ ] Logo at correct aspect ratio (no distortion)
- [ ] Hex codes verified against brand guidelines
- [ ] Photography matches brand image style
- [ ] No rogue gradients on flat brand (or vice versa)
- [ ] Voice and tone match brand verbal identity

---

**Delivery standard:** An asset that fails any item in Visual Hierarchy, Color, or Typography must be revised before delivery — these are not aesthetic preferences, they are performance factors with documented impact on conversion rates and brand equity.
