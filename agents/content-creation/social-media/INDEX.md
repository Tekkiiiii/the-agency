# Social Media Sub-team — Content Creation

**Platform-specific writers who produce native content for each social media channel.** Each writer masters their platform's culture, algorithm, format constraints, and audience expectations.

**Reports to**: Chief Content Officer / Content Director
**Model tier**: Sonnet

## Members

| Agent | Platform | What it produces |
|---|---|---|
| LinkedIn Writer | LinkedIn | Thought leadership posts, carousels, articles |
| Twitter/X Writer | Twitter/X | Tweets, threads, quote tweets, real-time copy |
| Instagram Writer | Instagram | Captions, Stories copy, Reels captions |
| TikTok Writer | TikTok | Captions, hook copy, text overlays, comments |
| Reddit Writer | Reddit | Posts, comments, AMA responses, community copy |
| Threads Writer | Threads | Conversational posts, hot takes, opinion threads |
| Facebook Writer | Facebook | Page posts, group content, event descriptions |
| Discord Writer | Discord | Server announcements, channel descriptions, bot messages |
| YouTube Writer | YouTube | Titles, descriptions, community posts, thumbnail text |
| Pinterest Writer | Pinterest | Pin titles/descriptions, board descriptions, idea pins |
| Quora Writer | Quora | Expert answers, Spaces posts, thought leadership |
| Telegram Writer | Telegram | Channel posts, group announcements, broadcast messages |
