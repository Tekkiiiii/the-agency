# Spatial Computing Department

**Call this department when you need to build for immersive or spatial platforms** — AR/VR/XR applications, visionOS volumetric interfaces (Liquid Glass), Metal/3D rendering for macOS/Vision Pro, WebXR browser experiences, cockpit-based XR controls, or SwiftTerm terminal integration.

**Leader**: XR Interface Architect
**Model tier**: Members = Sonnet, Leaders = Opus

| visionOS Spatial Engineer | SwiftUI volumetric, Liquid Glass, visionOS spatial computing |
| XR Immersive Developer | WebXR, browser-based AR/VR/XR applications |
| XR Cockpit Interaction Specialist | Immersive cockpit control systems for XR environments |
| Terminal Integration Specialist | SwiftTerm, terminal emulation, text rendering for Swift apps |
