## 2026-05-22 — Day 4 Hermes R1→R2: structural move validated, load shift caught

The R1 structural fix (4.31a/b → 4.02d/e) delivered exactly the predicted +5 points (+79→84). Forward-moving content into Section A works when the content is high-value positioning — but it also concentrates cognitive load before the first learner action. Watch for this side-effect: moving concept slides earlier can fix one momentum problem while creating a new passive-run problem if no engagement micro-moment is added.

Calibration note: a 8-slide passive run in a live remote session with CEO audience is a HIGH finding even when the content quality is good. Energy management matters more for this audience than for technical learners.

Forward anchors ("you'll see this at section X") are a cheap fix for concepts introduced before their experiential anchor — add them by default when any advanced feature is taught before learners have the tool installed.

Section checkpoints should exist no more than every 8-10 slides in the second half of a session (when energy is lowest). Map checkpoint gaps explicitly on second-pass reviews.
