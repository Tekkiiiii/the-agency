# Engineering Department

**Call this department when you need to write, review, or architect software systems** — backend services, APIs, databases, frontend UIs, mobile apps, DevOps pipelines, AI/ML integrations, security reviews, or firmware. Also for incidents, technical writing, and blockchain/smart contracts.

**Leader**: Backend Architect
**Model tier**: Members = Sonnet, Leaders = Opus

| AI Engineer | ML model development, deployment, data pipelines, AI-powered features |

| DevOps Automator | CI/CD pipelines, infra automation, cloud operations |

## Skills

These skills are loaded as process gates by dept members:
- **backend** — system design, API architecture, DB optimization
- **frontend** — UI implementation, responsive design, accessibility
- **security** — threat modeling, secure coding, vulnerability assessment
