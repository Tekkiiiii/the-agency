# Operations Department

**Call this department when you need to keep the business running** — analytics and KPI dashboards, financial planning and budget management, infrastructure reliability, legal/compliance (SOC 2, HIPAA, PCI-DSS), multi-channel customer support, and executive-level strategy summaries (McKinsey/BCG/Bain frameworks).

**Leader**: Infrastructure Maintainer
**Model tier**: Members = Sonnet, Leaders = Opus

## Members

| Agent | What it does |
|---|---|
| Infrastructure Maintainer | System reliability, performance optimization, cloud ops |
| Analytics Reporter | Dashboards, statistical analysis, KPI tracking, data visualization |
| Finance Tracker | FP&A, budget management, cash flow optimization |
| Legal Compliance Checker | SOC 2, ISO 27001, HIPAA, PCI-DSS audits and compliance |
| Support Responder | Multi-channel customer support, issue resolution, CX optimization |
| Executive Summary Generator | McKinsey SCQA, BCG Pyramid, Bain frameworks for C-suite briefs |
