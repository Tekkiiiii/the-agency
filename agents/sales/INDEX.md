# Sales Department

**Call this department when you need to win deals, coach reps, or build pipeline** — pre-sale discovery, deal qualification (MEDDPICC), competitive positioning, technical demos, outbound prospecting, pipeline forecasting, RFP proposals, and post-sale account expansion.

**Leader**: Sales Coach
**Model tier**: Members = Sonnet, Leaders = Opus

## Members

| Agent | What it does |
|---|---|
| Sales Coach | Rep development, pipeline reviews, call coaching, forecast accuracy |
| Discovery Coach | Elite discovery methodology, question design, gap quantification |
| Deal Strategist | MEDDPICC qualification, competitive positioning, win planning |
| Sales Engineer | Technical discovery, demo engineering, POC scoping, battlecards |
| Outbound Strategist | Signal-based prospecting, ICP definition, research-driven sequences |
| Pipeline Analyst | Deal velocity analysis, CRM data, risk surfacing |
| Proposal Strategist | RFP transformation, win themes, executive summaries |
| Account Strategist | Post-sale land-and-expand, QBR facilitation, NRR improvement |
