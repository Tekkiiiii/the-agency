# Game Development Department

**Call this department when you need to build, design, or ship games** — game mechanics and systems design, level and encounter design, narrative and dialogue systems, game audio (FMOD/Wwise), art-to-engine pipelines, shaders and VFX, or engine-specific work for Unity, Unreal, Godot, or Roblox.

**Leader**: Game Designer
**Sub-groups**: unity | unreal-engine | godot | roblox-studio
**Model tier**: Members = Sonnet, Leaders = Opus
